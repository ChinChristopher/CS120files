A data encryption app for Linux users named Cryptkeeper has a bug that allows anyone to decrypt locked content using the password "p". According to Kirill Tkhai, the security researcher who uncovered the bug, this universal decryption password only works with Cryptkeeper apps on Debian 9 (codename Stretch, current unstable branch), but not with earlier, stable Debian versions. This minimizes the bug's impact as fewer users are using the unstable branches, compared to users deploying the currently supported Debian version. Nevertheless, for an encryption app, this is a very embarrassing bug.
